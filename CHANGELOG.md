# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-25

### Added
- Initial release of Gotcha - Farcaster Growth Insights
- Wallet connection via Reown (WalletConnect)
- Farcaster profile integration using Neynar API
- Active follower analytics with activity scoring
- Time-based filtering (7, 14, 30 days)
- Cartoon-friendly UI with vibrant colors
- Stats cards for total followers, active followers, and activity rate
- Follower cards with activity badges
- Responsive design for mobile, tablet, and desktop
- Comprehensive documentation
- Vercel deployment configuration
- One-click deploy button

### Fixed
- Avatar image fallback for broken images

### Performance
- Optimized data fetching with useCallback
- API response caching

### Documentation
- README with quick start guide
- Project summary and architecture docs
- UI design system documentation
- Deployment guides for multiple platforms
- Vercel-specific deployment guide
